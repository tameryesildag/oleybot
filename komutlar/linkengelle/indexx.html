o!linkengelle {aç/kapa} {#kanal/tüm}

"o!linkengelle aç": mesajın yazıldığı kanalda link engellemeyi açar.
"o!linkengelle kapa": mesajın yazıldığı kanalda link engellemeyi kapar.
"o!linkengelle aç #genel": etiketlenen kanalda link engellemeyi açar.
"o!linkengelle kapa #genel": etiketlenen kanalda link engellemeyi kapar.
"o!linkengelle aç tüm": tüm yazı kanallarında link engellemeyi açar.
"o!linkengelle kapa tüm": tüm yazı kanallarında link engellemeyi kapar.
