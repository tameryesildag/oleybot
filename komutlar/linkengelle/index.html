o!linkengelle {aç/kapa} {#kanal/tüm}

Linkleri otomatik olarak silme özelliğini açıp kapamaya yarar. Bu özellik açıkken bot, içinde link olan bir mesaj gönderildiğinde otomatik olarak siler. "Mesajları Yönet" yetkisine sahip olanların mesajları silinmez.

o!linkengelle aç Komutun yazıldığı kanalda link engellemeyi açar.
o!linkengelle kapa Komutun yazıldığı kanalda link engellemeyi kapar.
o!linkengelle aç #genel Etiketlenen kanalda link engellemeyi açar.
o!linkengelle kapa #genel Etiketlenen kanalda link engellemeyi kapar.
o!linkengelle aç tüm Tüm yazı kanallarında link engellemeyi açar.
o!linkengelle kapa tüm Tüm yazı kanallarında link engellemeyi kapar.
